# assets/

Déposez ici l'image de fond du héro sous le nom **`hero.jpg`**.

- Format recommandé : JPG ou WebP, paysage (ratio ~16:9), ~1920×1080 px, < 400 Ko.
- Sujet conseillé : intérieur rénové / mur placo / parquet, lumière naturelle,
  espace dégagé à gauche pour laisser respirer le titre.
- Tant que `hero.jpg` est absent, un dégradé de secours s'affiche automatiquement
  (aucun visuel cassé).

Pour changer le nom ou le chemin du fichier, modifier la variable CSS
`--hero-photo` dans `index.html` (`:root`).
