Image de fond actuelle : photo **Unsplash** (intérieur épuré, murs lisses
finition placo, parquet), téléchargée et hébergée **en local** pour garder le
site autonome (aucune dépendance CDN au runtime).

- Format livré : JPEG 1600 px, ~330 Ko.

## Remplacer par une vraie photo Ak_Home

Pour utiliser un cliché du compte `@ak_home72`, remplacez simplement le fichier
`assets/hero.jpg` (paysage ou portrait, l'image est recadrée automatiquement en
`cover`). Idéalement : intérieur rénové, espace dégagé à gauche pour le titre,
lumière naturelle, < 400 Ko.

Pour changer le nom ou le chemin du fichier, modifier la variable CSS
`--hero-photo` dans `index.html` (`:root`).
